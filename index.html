Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node. You can download Vs code from this link https://code.visualstudio.com/ you will need to run the setup file after installation and it will tell you how to download.

Console log is used to test codes if they are working or not like if you initialised a variable you can see the variable's value using console log. Inorder to access the console screen just press Cmd + Option + J (on a Mac) or Ctrl +Shift +J (on Windows). If you want to print your name on the console screen in js file type console.log("yourname"); .
